Como o Sistema Funciona

Uma arquitetura distribuída desenhada para não perder nenhum dado e alertar no momento exato.

Coleta de Dados

Sensores e entradas manuais enviam leituras de nível e condições dos rios continuamente.

Distribuição

Os dados trafegam pela rede distribuída em filas de eventos, garantindo a entrega.

Monitoramento

Um servidor Host consolida os dados em tempo real, enquanto Réplicas ficam em standby.

Alertas por Nível

Caso o nível ultrapasse limites críticos, alertas preventivos são disparados para a central.
